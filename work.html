MyFlix App

MyFlix is a web application built with React that allows users to explore and enjoy a vast collection of movies. With a sleek, responsive interface powered by React Bootstrap and React Router, MyFlix offers seamless navigation and an engaging movie browsing experience. The app is fully deployed on Netlify and connected to a MongoDB-based backend hosted on Heroku.
